打开方式：
1.打开server.js服务器文件输入命令启动node(node.server)
2.打开pets项目文件夹的index.html
3.首页地址：http://localhost:7666/index.html
4.通过点击头部的来注册登陆账号
5.通过点击轮播图进入商品列表
6.通过点击导航栏里的购物车进入购物车（需要登录）
7.通过点击商品列表的某个商品进入商品详情页
8.源文件pets
以实现部分功能：
1.轮播图
2.登陆注册
3.商品列表
4.商品详情页
5.购物车
6.注销
